How to Generate Turtle Images Locally

Follow these steps to generate the cute baby turtle images on your local machine:

Step 1: Install Python

You'll need Python installed on your computer. If you don't have it yet:

  1. Go to https://www.python.org/downloads/
  2. Download the latest version for Windows
  3. Run the installer and make sure to check "Add Python to PATH"

Step 2: Set up your project

Open Command Prompt and run these commands:

cd C:\Users\deployyyyyyyyyyyyyyy\Desktop\New folder
pip install google-genai==0.1.0 Pillow

Step 3: Create a .env file with your Gemini API key

Create a new file named ".env" in the same folder and add this line:

GEMINI_API_KEY=AIzaSyC8FuTNC3FxLs0Qx2ciRoLwxjOrLGqOB5A

Step 4: Generate the images

Run the following command to generate the turtle images:

python generate_image.py

Success!

After running the script, you should have:

  • A turtle_images folder containing multiple PNG images
  • An output_story.txt file with the story
  • An output_image.png file (the last image)

What's happening behind the scenes

When you run the script:

  1. It connects to the Google Gemini API using your provided API key
  2. It sends a prompt asking for a story about a cute baby turtle with images
  3. The API generates both the story text and multiple images
  4. The script saves each image and the story text to your local machine

Each time you run it, you'll get a different story and different images!
